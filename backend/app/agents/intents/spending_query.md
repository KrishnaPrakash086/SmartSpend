# Intent: spending_query

## Description
User wants to know how much they have SPENT in a period or category.

## Examples
- how much did I spend today
- total spending this month
- what did I spend this week
- my spending breakdown
- how much on food this month
- spending history
- total expenses today
- what was my spending last month

## Keywords
spending, spent, spend, total expenses, how much, this month, this week, today, last month
