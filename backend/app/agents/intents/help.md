# Intent: help

## Description
User is asking what the assistant can do, seeking guidance, or requesting instructions.

## Examples
- help
- help me
- what can you do
- who are you
- what is this
- how does this work
- what are your features
- show me options
- what should I do
- guide me
- how to use

## Keywords
help, features, guide, options, instructions, capabilities, tutorial
